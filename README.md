# Chrome-extension

<video src="./Video Project 1.mp4"></video>

复制下面命令到终端下载
```bash
git clone https://github.com/YiYamimimi/Chrome-extension.git
```

把`dist`文件夹拖到浏览器的扩展程序页面中。
1. 打开 Chrome 浏览器。
2. 点击 `更多工具`。
3. 点击 `扩展程序`。
4. 点击 `加载已解压的扩展程序`。
5. 选择 `dist` 文件夹。
6. 插件就会加载到 Chrome 浏览器中。
7. 点击 `扩展程序`。

该插件提供实时问答、核心主题、字幕可视化等辅助功能，帮助用户更方便地学习和理解视频内容。
